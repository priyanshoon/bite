# Bite

![head](./assets/head.png) 

A dead simple static site generator that converts raw markdown to raw html format.

# Prerequisite 

- Python >= 3.12
- knowledge of cli

# Installation

Clone the repository -
```zsh
git clone https://github.com/priyanshoon/bite
```

# Usage

- Your markdown should be inside `content` directory, I have pushed some example files in there
- All the images and styles should be in `static` directory.

Once you done this, run the below command to generate the desire htmls

```zsh
./main.sh
```
